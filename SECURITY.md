# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :bug修复直到下下个大版本发布: |
| dev   | :发布当天的bug修复:                |


## Reporting a Vulnerability

建议使用以下模板
version=使用的版本
Short_description=(例如:Unauthorized_access_Error)
descriptio=复现步骤
